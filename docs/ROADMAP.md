# Roadmap

## Implemented in 0.2

- 80 tools across CLI, filesystem, network, Git, archives, OS administration, and computer use.
- Mouse, keyboard, window, application, monitor and screenshot adapters.
- Accessibility snapshots and local OCR/text-click workflows.
- Observe-act-observe autonomous action primitives.
- OpenAI Secure MCP Tunnel setup scripts.
- Optional authenticated Streamable HTTP transport.
- Cross-platform CI and MCP tool-discovery integration test.

## Implemented on main for 0.3

- Native file-watch sessions with bounded event retention, cursor pagination, policy enforcement, and lifecycle cleanup.

## 0.3 remaining

- Browser-specific CDP/Playwright adapter alongside generic computer use.
- Rich binary document adapters for PDF, DOCX, XLSX, images, audio/video metadata, and databases.
- Stronger Wayland backends and compositor-specific capability adapters.
- macOS semantic element invoke and richer Linux AT-SPI action support.
- Signed downloadable binaries and installers.

## 0.4

- Device pairing UI and capability-grant dashboard.
- Principal-aware authorization policies for direct HTTP mode.
- OAuth 2.1 authorization-server reference deployment.
- Per-tool approval modes and one-time grants.
- End-to-end audit correlation IDs and structured event export.
- Multi-device registry/routing for one ChatGPT app.

## 1.0 criteria

- Stable schemas and migration policy.
- Native integration tests on physical/virtual Windows, macOS, X11, and representative Wayland compositors.
- Signed releases, SBOM, provenance attestations, and reproducible builds.
- Independent security review of safe-mode boundaries and remote transport.
- Complete installer/uninstaller and startup-service lifecycle.
